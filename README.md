# Rockbuster Stealth Data Analysis
Rockbuster Stealth LLC is a fictitious movie rental company introduced as part of CareerFoundry data analystics course. Data analysis using PostgreSQL will  done using Rockbuster Stealth database. 
## Objective
Aim of this project is to provide data-driven answers for series of business questions to help build launch strategies for new online video games.
